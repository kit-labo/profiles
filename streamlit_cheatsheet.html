🎨 Streamlit チートシート

小中学生のためのプログラミングサポート

📚 使い方

  • 各コードブロックの「コピー」ボタンを押すとコードがコピーできます
  • ❓マークの部分は、自分で考えて入力してね!
  • ヒントを参考にしながら、どんな値を入れるか考えてみよう
  • まずは小さく試して、動いたら少しずつ変えてみよう!
⚙️ 1. 基本設定
ライブラリの読み込み
import streamlit as st
💡 これは必ず最初に書こう! Streamlitを使えるようにする魔法の呪文だよ
ページの設定
st.set_page_config(page_title=❓, page_icon=❓)
💡 ヒント: page_title には「"」で囲んだページのタイトル、page_icon には絵文字を入れよう
例: page_title="私のアプリ", page_icon="🌟"
✅ 完成例: st.set_page_config(page_title="おみくじ", page_icon="🎴")
📝 2. 文字を表示する
タイトル
st.title(❓)
💡 大きな見出しだよ!
「"」で囲んだ文字を入れよう
ヘッダー
st.header(❓)
💡 中くらいの見出しだよ!
サブヘッダー
st.subheader(❓)
💡 小さめの見出しだよ!
普通の文
st.write(❓)
💡 一番よく使う! 普通の文を表示するよ
シンプルなテキスト
st.text(❓)
💡 シンプルな文字だけを表示
マークダウン
st.markdown(❓)
💡 **太字**や*斜体*が使えるよ!
💬 3. メッセージボックス
成功メッセージ
st.success(❓)
💡 緑色の成功メッセージ!
「やったね!」みたいな時に使おう
情報メッセージ
st.info(❓)
💡 青色の情報メッセージ!
お知らせに使おう
警告メッセージ
st.warning(❓)
💡 黄色の警告メッセージ!
注意してほしい時に使おう
エラーメッセージ
st.error(❓)
💡 赤色のエラーメッセージ!
間違いやエラーを表示する時に使おう
🎮 4. ボタンと入力
ボタン
if st.button(❓): # ここに押された時の処理を書く
💡 ボタンを作るよ! ボタンに表示する文字を入れよう
if の次の行は、半角スペース4つ分 下げて書くよ
✅ 完成例:
if st.button("クリック!"):
    st.success("押されたよ!")
テキスト入力
name = st.text_input(❓)
💡 文字を入力してもらうよ!
入力欄の上に表示する説明文を入れよう
入力された文字は name という変数に保存されるよ
✅ 完成例: name = st.text_input("あなたの名前は?")
数値入力
number = st.number_input(❓, min_value=❓, max_value=❓, value=❓)
💡 数字を入力してもらうよ!
最初の❓: 説明文
min_value: 最小の値、max_value: 最大の値
value: 最初に表示される数字
✅ 完成例: number = st.number_input("好きな数字", min_value=1, max_value=100, value=50)
スライダー
value = st.slider(❓, ❓, ❓, ❓)
💡 スライダーで数を選んでもらうよ!
1つ目の❓: 説明文
2つ目の❓: 最小値、3つ目の❓: 最大値
4つ目の❓: 最初の位置
✅ 完成例: age = st.slider("年齢は?", 0, 100, 10)
セレクトボックス
choice = st.selectbox(❓, [❓, ❓, ❓])
💡 選択肢から選んでもらうよ!
1つ目の❓: 説明文
[ ] の中に選択肢を「"」で囲んで入れよう
✅ 完成例: choice = st.selectbox("好きな色は?", ["赤", "青", "緑"])
ラジオボタン
choice = st.radio(❓, [❓, ❓, ❓])
💡 選択肢をボタンで表示するよ!
使い方は selectbox と同じだよ
チェックボックス
if st.checkbox(❓): # チェックされた時の処理
💡 チェックボックスを作るよ!
チェックされたら True、されてなければ False
📐 5. レイアウト
区切り線
st.divider()
💡 横線を引くよ! セクションを分けたい時に使おう
カラム(2列)
col1, col2 = st.columns(2) with col1: # 左側の内容 with col2: # 右側の内容
💡 画面を2つに分けるよ!
数字を変えれば3列、4列もできるよ
カラム(3列)
col1, col2, col3 = st.columns(3) with col1: # 左側 with col2: # 真ん中 with col3: # 右側
💡 画面を3つに分けるよ!

pandasのDataFrameを入れよう
棒グラフ
st.bar_chart(❓)

DataFrameやリストを入れよう

面グラフ

🎉 7. アニメーション
風船
st.balloons()
💡 お祝いの風船が飛ぶよ!
読み込み中
with st.spinner(❓): # 処理を書く
💡 くるくる回るアニメーション!
「"」で囲んだメッセージを入れよう
✅ 完成例:
with st.spinner("計算中..."):
    time.sleep(2)
⭐ 8. よく使うパターン
基本的なアプリの構造
import streamlit as st st.set_page_config(page_title=❓, page_icon=❓) st.title(❓) st.write(❓) if st.button(❓): st.success(❓)
💡 これが基本の形! ここから始めてみよう
入力があるかチェック
name = st.text_input(❓) if name: st.write(f"こんにちは、{name}さん!") else: st.warning("名前を入力してね!")
💡 入力があるか確認するパターン!
if name: で入力があるかチェックできるよ
条件分岐(if-elif-else)
value = st.slider(❓, ❓, ❓, ❓) if value < ❓: st.info(❓) elif value < ❓: st.warning(❓) else: st.success(❓)
💡 値によって違う処理をするパターン!
if: もし〜なら
elif: そうじゃなくて〜なら
else: それ以外なら
🐍 9. Pythonの基本も覚えよう
変数に保存
name = "たろう" age = 12
💡 値を保存しておくよ!
文字は「"」で囲む、数字はそのまま
リスト
colors = ["赤", "青", "緑"]
💡 複数の値をまとめるよ!
[ ] で囲んで、,(カンマ)で区切る
f文字列
name = "たろう" st.write(f"こんにちは、{name}さん!")
💡 変数を文字の中に入れるよ!
f"文字{変数}文字" という形で使う
